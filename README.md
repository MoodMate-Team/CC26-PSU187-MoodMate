# CC26-PSU187-MoodMate
Dashboard MoodMate (Capstone CC26-PSU187) berbasis Streamlit &amp; Plotly menganalisis tren emosi pengguna. Lewat pendekatan *Painkiller*, platform AI ini meningkatkan *self-awareness* melalui klasifikasi teks otomatis ke emosi spesifik (Joy, Sadness, Anger, Fear, Love, Surprise) untuk deteksi dini stres dan rekomendasi aktivitas nyata.
